Mtas
The mtas section occurs within the response, and describes mtas results that should be included in the response. The mtas section is of the form 
{
  "stats" : <mtas statistics>,
  "document" : <mtas document>,
  "kwic" : <mtas kwic>,
  "list" : <mtas list>,
  "termvector" : <mtas termvector>,
  "facet" : <mtas facet>,
  "group" : <mtas group>,
  "prefix" : <mtas prefix>,
  "collection" : <mtas collection>,
  "version" : <mtas version>,
}
