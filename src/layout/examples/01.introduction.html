Introduction
Each search request to the broker is of the form
{
  "configuration" : <string>,
  "filter" : <filter>,
  "condition" : <condition>,
  "response" : <response>,        
  "sort" : [ <sortitem>, <sortitem>,... ],
  "debug" : <string>,
  "cache" : <boolean>,
}
Here only response is obligatory, configuration, filter, condition, sort, debug and cache are optional.
{ "response": { "documents": { "start": 0, "rows": 10, "fields": [ "::uniqueKey::" ] }} }
 Get unique key ::uniqueKey:: for the first 10 documents.
