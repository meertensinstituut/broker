Mtas Collection
The mtas collection section occurs within the mtas section, and describes collection items that should be included in the response for the set of documents matching condition and filter. The mtas collection section is an array containing one or multiple of the following items
  • create : create a (new) collection
  • post : post a (new) collection
  • list : list available collections
  • check : check collection
  • import : import collection
  • delete : delete collection
  • empty : remove all collections
  • get : get collection
Create
An item in the mtas collection to create is of the form 
{
  "action" : "create",
  "field" : <string>,
  "key" : <string>,
  "id" : <string>,
}
{ "condition" : { "type": "equals", "field" : "::exampleFieldText(fieldName)::", "value" : "::exampleFieldTextValue0(word)::"},"response": { "mtas": { "collection" : [ { "key": "example create", "action": "create", "field": "::exampleFieldString(fieldName)::" }] }}}
 Create collection by collecting occuring values from ::exampleFieldString(fieldName):: for the selected set of documents.
Post
An item in the mtas collection to post is of the form 
{
  "action" : "post",
  "post" : <string>,
  "key" : <string>,
  "id" : <string>,
}
{ "response": { "mtas": { "collection" : [ { "key": "example post", "action": "post", "post": ["een", "twee", "drie,vier"], "id": "1234567890" }] }}}
 Post collection "een", "twee", "drie,vier".
List
An item in the mtas collection to get a list is of the form 
{
  "action" : "list",
  "key" : <string>,
}
{ "response": { "mtas": { "collection" : [ { "key": "example list", "action": "list" }] }}}
 Post collection "een", "twee", "drie,vier".

{
  "action" : "check",
  "id" : <string>,
  "key" : <string>,
}
{ "response": { "mtas": { "collection" : [ { "key": "example check", "action": "check", "id": "1234567890" }] }}}

Import
An item in the mtas collection to import is of the form 
{
  "action" : "import",
  "configuration" : <string>,
  "collection" : <string>,
  "key" : <string>,
  "id" : <string>,
}
{ "configuration": "::configuration1(config2)::", "response": { "mtas": { "collection" : [ { "key": "example import", "action": "import", "configuration": "::configuration0(config1)::", "collection": "1234567890" }] }}}
 Import collection by collecting occuring values from ::exampleFieldString(fieldName):: for the selected set of documents.
Delete
An item in the mtas collection to delete is of the form 
{
  "action" : "delete",
  "id" : <string>,
  "key" : <string>,
}
{ "response": { "mtas": { "collection" : [ { "key": "example delete", "action": "delete", "id": "1234567890" }] }}}
 Delete collection by identifier.
Empty
An item in the mtas collection to empty is of the form 
{
  "action" : "empty",
  "key" : <string>,
}
{ "response": { "mtas": { "collection" : [ { "key": "example empty", "action": "empty" }] }}}
 Remove all collections.
Get
An item in the mtas collection to get is of the form 
{
  "action" : "get",
  "id" : <string>,
  "key" : <string>,
}
{ "response": { "mtas": { "collection" : [ { "key": "example get", "action": "get", "id": "1234567890" }] }}}
 Get collection with provided identifier.
