Configuration
The configuration used for a request defines which solr core(s) are used to process the requests. Since the solr schema for these cores describe the available fields and their properties, the broker can try to derive which configurations can be used for a request.

If no specific configuration is defined, the broker will try to find and use the first configuration containing all necessary fields
