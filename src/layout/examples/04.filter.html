Filter
A filter can be used to restrict the super set of documents that can be returned, without influencing score. The filter can be declared as a filteritem or as an array of filteritems when multiple filters are needed.
Filteritem
The filteritem is of the form 
{
  "condition" : <condition>,
  "tag" : <string>,
}
Each filteritem should always have a condition. The tag is a string referred to when a filter has to be ignored when computing facets.
{ "filter" : {"condition" : { "type": "equals", "field" : "::exampleFieldText(fieldName)::", "value" : "::exampleFieldTextValue0(word)::"}}, "response": { "documents": { "start": 0, "rows": 10, "fields": [ "::uniqueKey::" , "::exampleFieldText(fieldName)::"] }} }
 Get unique key and field ::exampleFieldText(fieldName):: for the first 10 documents where field ::exampleFieldText(fieldName):: contains a term equal to '::exampleFieldTextValue0(word)::' by using a filter.
Multiple filters
To use multiple filters, define the filter as an array of filteritems 
[{
  "condition" : <condition>,
  "tag" : <string>,
},{
  "condition" : <condition>,
  "tag" : <string>,
}]
{ "filter" : [{"condition" : { "type": "equals", "field" : "::exampleFieldText(fieldName)::", "value" : "::exampleFieldTextValue0(word)::"}},{"condition" : { "type": "equals", "field" : "::exampleFieldText(fieldName)::", "value" : "::exampleFieldTextValue1(word)::"}}], "response": { "documents": { "start": 0, "rows": 10, "fields": [ "::uniqueKey::" , "::exampleFieldText(fieldName)::"] }} }
 Get unique key and field ::exampleFieldText(fieldName):: for the first 10 documents where field ::exampleFieldText(fieldName):: contains a term equal to '::exampleFieldTextValue0(word)::' and field ::exampleFieldText(fieldName):: contains a term equal to '::exampleFieldTextValue1(word)::' by using two filters.
