Mtas Prefix
The mtas prefix section occurs within the mtas section, and describes mtas prefix that should be included in the response. The mtas prefix section is of the form 
{
  "field" : <string>,
  "key" : <string>,
}
{ "response": { "mtas": { "prefix" : [{ "key": "list of prefixes", "field": "::exampleFieldMtas(fieldName)::" }] } } }
 Get prefixes for field ::exampleFieldMtas(fieldName)::.
