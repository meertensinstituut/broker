Mtas Page
The mtas page section occurs within the mtas section, and describes page results that should be included in the response for the listed documents. The mtas page section is of the form 
[
  {
    "field" : <field>,
    "start" : <integer>,
    "end" : <integer>,
    "key" : <string>,
    "prefix" : <string>,
  },... 
]
{ "condition" : { "type": "cql", "field" : "::exampleFieldMtas(fieldName)::", "value" : "[]"}, "response": { "documents": { "start": 0, "rows": 2, "fields": [ "::uniqueKey::", "::exampleFieldText(fieldName)::" ] }, "mtas": { "page" : [{ "field": "::exampleFieldMtas(fieldName)::", "start": 0, "end": 100, "prefix": "::exampleFieldMtasWordPrefix(prefix1)::,::exampleFieldMtasLemmaPrefix(prefix2)::,::exampleFieldMtasPosPrefix(prefix3)::" }] } } }
 Get page from position 0 until 100 with prefixes ::exampleFieldMtasWordPrefix(prefix1)::, ::exampleFieldMtasLemmaPrefix(prefix2):: and ::exampleFieldMtasPosPrefix(prefix3):: in ::exampleFieldMtas(fieldName)::.
